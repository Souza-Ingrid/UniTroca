# 📚 UniTroca

Plataforma web para troca, doação e compartilhamento de materiais acadêmicos entre estudantes.

O projeto foi desenvolvido com foco em resolver um problema real: o alto custo de materiais acadêmicos e o desperdício de itens ainda úteis.

---

## 🚀 Demonstração

🔗 https://unitroca.vercel.app

---

## 💡 Funcionalidades

* 📌 Cadastro de itens (livros, resumos, calculadoras, etc.)
* 📋 Listagem dinâmica dos itens cadastrados
* 🔍 Busca por nome do item
* 🗂️ Filtro por categoria
* 🎨 Interface responsiva com Bootstrap
* ⚡ Atualização dinâmica da interface sem recarregar a página

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3 (Bootstrap)
* JavaScript (DOM manipulation)
* Vercel (deploy)

---

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto, foram aplicados conceitos importantes como:

* Manipulação do DOM com JavaScript
* Organização de código em funções reutilizáveis
* Estruturação de interface responsiva
* Lógica de filtros e busca em tempo real
* Fluxo de dados entre interface e lógica
